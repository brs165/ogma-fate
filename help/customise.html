Customise

Making Ogma work for your table

Ogma adapts to different experience levels, play styles, and table setups. Here's how to tune it to your needs.

Result Reference Panel

Every generator result shows a reference panel directly below it. Three tabs give you different levels of context — no configuration required, all three are always available:

TabWho it's forWhat it shows
What · For GMDefault — all GMsWhat the generator produces, running tips, invoke and compel examples
🌿 New here?New to tabletop RPGs or FatePlain-language explanation of the mechanic and how it works at the table
⚔ D&D?Converting from D&D 5e / PathfinderSide-by-side comparison of Fate and D&D mechanics — aspects vs ability scores, stress vs HP, fate points vs inspiration

Skills

Ogma uses the 19 default Fate Condensed skills (Academics, Athletics, Burglary, Contacts, Crafts, Deceive, Drive, Empathy, Fight, Investigate, Lore, Notice, Physique, Provoke, Rapport, Resources, Shoot, Stealth, Will). If your campaign uses a custom skill list or approaches (from Fate Accelerated), the generated skill pyramids will use FCon defaults. The Fate System Toolkit has guidance on building custom skill lists.

Table Manager — customising results

Open the Table Manager with the ⊞ button in the action bar. It shows every table that feeds the current generator.

  • Exclude an entry by toggling the checkbox — it won't appear in future rolls. Useful when you've already used that NPC name or faction descriptor.
  • Lock an entry to always use it — roll any number of times and that entry will be the one that appears for that slot. Useful for fixed character names.
  • Add custom entries — type your own entry and add it to the table. It participates in generation alongside the built-in content.

Table customisations are saved to localStorage and persist across browser restarts.

Universal table merge

Ogma has a set of Universal tables — genre-neutral content that works in any campaign. The Universal Merge toggle (in the sidebar Preferences panel) controls whether these tables supplement the campaign-specific ones:

  • On (default) — universal entries are added to the campaign pool. More variety, slightly less world-specific flavour.
  • Off — only campaign-specific entries. Maximum thematic consistency.

Session Zero wizard

The Prep Wizard (⚡ Prep in the topbar) is a 5-step guided flow that gets you session-ready in minutes: choose a world, describe your players, generate a seed, build a scene, and create an NPC. Each step lets you reroll and the output saves automatically to IndexedDB for import into the Run surface.

For collaborative character creation with your players, use the Character Creation Wizard — it walks a group through the Phase Trio, skills, stunts, and refresh.

D&D Transition guide

The D&D Transition guide explains every major mechanical difference between Fate Condensed and D&D — side by side, in plain language. Available from the wiki sidebar on any Help & Wiki page.

Theme and text size

Press the ◑ button in the topbar to toggle between dark and light mode. Your preference is saved to localStorage.

Text size can be adjusted via the ⊞ Table Manager panel or the Preferences panel in the sidebar. Three levels are available.

Keyboard shortcuts

Press ? to open the full keyboard shortcuts reference. All shortcuts are single-key (no modifier required) and are blocked when a text input is focused.
