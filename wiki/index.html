Quick reference — keyboard shortcuts

KeyAction
SpaceRoll the current generator
GCycle to next generator
PPin current result to History
ZUndo last pin
IInspiration mode — roll 3, pick one
CCopy result as Markdown
LCopy shareable link for this result
VPush result to projector / second screen
?Open keyboard shortcuts panel
EscClose any open panel or sidebar

About Ogma

Ogma — On-demand Generator for Masterful Adventures — is an independent fan project built on the Fate Condensed SRD (CC BY 3.0). It runs entirely in your browser with no server, no account, and no internet connection required after the first load.

7 campaign worlds. 16 generators. 112 generator/campaign combinations. 9,750+ table entries.
