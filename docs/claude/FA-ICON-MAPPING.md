## IMPLEMENTATION NOTES

1. Generator icons in shared.js are currently plain text strings ("◈").
   Switching to FA requires changing how icons render — from `{gen.icon}` text
   to `<i class={gen.iconClass}></i>`. This touches every component that
   reads gen.icon. Consider adding an `iconClass` field alongside `icon`.

2. Stress checkboxes (☐, 19 uses) are deeply integrated into StressRow.svelte
   and PlayerRow.svelte. These are functional, not decorative — replacing with
   FA `fa-square` / `fa-square-check` changes interaction patterns.

3. The ☀️ emoji for light mode toggle is a raw emoji, not an HTML entity.
   Replace with `fa-solid fa-sun`.

4. Total scope: ~180 individual replacements across ~40 files.
   Estimated effort: 2 sessions.
