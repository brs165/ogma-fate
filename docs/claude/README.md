# docs/claude/

This directory contains reference materials for AI-assisted development sessions on Ogma.

Ogma is built with Claude as a development partner. These files capture the project knowledge, conventions, and prompts that make AI sessions effective and consistent.

## Files

| File | When to load |
|------|-------------|
| [BOOTSTRAP.md](./BOOTSTRAP.md) | **Every session.** Project overview, file map, engineering rules, QA commands, architecture summary. |
| [CONVENTIONS.md](./CONVENTIONS.md) | Engineering and accessibility sessions. WCAG/HIG rules, working conventions, hard lessons, role-specific session prompts. |
| [WORLD-VOICES.md](./WORLD-VOICES.md) | Content and world-building sessions. Tone, voice, and creative DNA for each campaign world. |

## How to start a session

Point Claude at `docs/claude/BOOTSTRAP.md`. That file is self-contained and will orient a new session in full. Load the other files as needed for specialised work.
